Typography & Basic Styling

This page demonstrates how the CSS Box Model affects layout. Each box you see is a block-level element with its own content, padding, border, and margin. Try toggling the .debug class in the CSS to visualize boundaries.

Tip: Padding increases space inside a box around its content, while margin creates space outside the box between neighbors. Borders sit between padding and margin.
